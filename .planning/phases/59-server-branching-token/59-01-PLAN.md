**1. Add `resolve_active_path` function to `crates/hydeclaw-core/src/db/sessions.rs`:**

Create `pub async fn resolve_active_path(db: &PgPool, session_id: Uuid, leaf_message_id: Option<Uuid>) -> Result<Vec<MessageRow>>`.

Logic:
- If `leaf_message_id` is `Some(id)`, delegate to existing `load_branch_messages(db, session_id, id)`.
- If `leaf_message_id` is `None`, find the "latest leaf" automatically:
  1. Query all messages for the session: `SELECT id, parent_message_id FROM messages WHERE session_id = $1`
  2. Build a set of all `parent_message_id` values (non-null)
  3. Find messages whose `id` is NOT in the parent set — these are leaves
  4. Among the leaves, pick the one with the latest `created_at`
  5. If there are no branch messages at all (all parent_message_id are NULL), fall back to `load_messages(db, session_id, None)` — regular flat history
  6. Otherwise call `load_branch_messages(db, session_id, latest_leaf_id)`

**3. Add `api_active_path` handler in `sessions.rs`:**

```rust
#[derive(Deserialize)]
pub(crate) struct ActivePathQuery {
    leaf: Option<uuid::Uuid>,
}

/// GET /api/sessions/{id}/active-path — resolve the linear message chain for display.
pub(crate) async fn api_active_path(
    State(state): State<AppState>,
    Path(session_id): Path<uuid::Uuid>,
    Query(q): Query<ActivePathQuery>,
) -> impl IntoResponse {
    match sessions::resolve_active_path(&state.db, session_id, q.leaf).await {
        Ok(msgs) => {
            let messages: Vec<Value> = msgs.iter().map(|m| {
                json!({
                    "id": m.id,
                    "role": m.role,
                    "content": m.content,
                    "tool_calls": m.tool_calls,
                    "tool_call_id": m.tool_call_id,
                    "created_at": m.created_at.to_rfc3339(),
                    "agent_id": m.agent_id,
                    "feedback": m.feedback.unwrap_or(0),
                    "edited_at": m.edited_at.map(|t| t.to_rfc3339()),
                    "status": m.status,
                    "parent_message_id": m.parent_message_id,
                    "branch_from_message_id": m.branch_from_message_id,
                })
            }).collect();
            Json(json!({ "messages": messages })).into_response()
        }
        Err(e) => (StatusCode::INTERNAL_SERVER_ERROR, Json(json!({"error": e.to_string()}))).into_response(),
    }
}
```
