**1. OpenAI-compat handler (around line 216):**

**Import changes (line 15):**
- Replace `use tokio_stream::wrappers::UnboundedReceiverStream;` with `use tokio_stream::wrappers::ReceiverStream;`

**Capacity rationale:** 256 for OpenAI-compat (simple text deltas), 512 for main chat (richer event types including tool calls). At ~200 bytes per SSE event, 512 events = ~100KB buffer — acceptable memory ceiling per connection.

```rust
/// Parse CIDR subnets and return their gateway IPs (.1 address).
/// e.g. "172.17.0.0/16" -> "172.17.0.1"
fn get_docker_subnet_gateways(subnets: &[String]) -> Vec<String> {
    subnets.iter().filter_map(|cidr| {
        let ip_part = cidr.split('/').next()?;
        let octets: Vec<&str> = ip_part.split('.').collect();
        if octets.len() == 4 {
            // Replace last octet with 1 (gateway convention)
            Some(format!("{}.{}.{}.1", octets[0], octets[1], octets[2]))
        } else {
            None
        }
    }).collect()
}
```
