# Summarize MCP

URL and text summarization in Russian via LLM.

## Tools

### summarize_url
- **url** (required): URL to summarize
- **max_length** (optional): Maximum length in words (default: 500)

Fetches the page, extracts text (readability), summarizes via LLM.

### summarize_text
- **text** (required): Text to summarize
- **max_length** (optional): Maximum length in words (default: 500)
